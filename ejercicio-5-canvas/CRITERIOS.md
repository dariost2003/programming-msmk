# Criterios de Evaluacion - Canvas Interactivo

## Puntuacion total: 100 puntos

---

## Tarea 1: Sistema de dibujo libre (20 puntos)

| Criterio | Puntos |
|----------|--------|
| Se puede dibujar a mano alzada en el canvas | 5 |
| El trazo se renderiza en tiempo real mientras se dibuja | 5 |
| Color del trazo configurable desde la paleta | 4 |
| Grosor del trazo configurable desde el slider | 3 |
| Codigo limpio y documentado | 3 |

## Tarea 2: Creacion y seleccion de figuras (20 puntos)

| Criterio | Puntos |
|----------|--------|
| Crear rectangulos arrastrando en el canvas | 4 |
| Crear circulos arrastrando en el canvas | 4 |
| Click selecciona una figura (handles de seleccion visibles) | 5 |
| Agregar texto al canvas | 4 |
| Codigo limpio y documentado | 3 |

## Tarea 3: Arrastrar y reposicionar (20 puntos)

| Criterio | Puntos |
|----------|--------|
| Mover elementos arrastrando con el mouse | 6 |
| Movimiento fluido en tiempo real (sin saltos) | 5 |
| Distinguir correctamente entre click (seleccionar) y drag (mover) | 4 |
| Snap-to-grid funcional | 2 |
| Codigo limpio y documentado | 3 |

## Tarea 4: Flechas y conectores (20 puntos)

| Criterio | Puntos |
|----------|--------|
| Dibujar flechas entre dos puntos cualesquiera | 4 |
| Punta de flecha (triangulo) visible y orientada correctamente | 3 |
| Conectar flechas a elementos existentes (source_id/target_id) | 5 |
| Las flechas se actualizan automaticamente al mover elementos conectados | 5 |
| Codigo limpio y documentado | 3 |

## Tarea 5: Widgets interactivos (20 puntos)

| Criterio | Puntos |
|----------|--------|
| Botones funcionales colocados sobre el canvas | 4 |
| Sliders funcionales colocados sobre el canvas | 4 |
| Campos de texto funcionales colocados sobre el canvas | 4 |
| Los widgets son arrastrables (se pueden reposicionar) | 5 |
| Codigo limpio y documentado | 3 |

---

## Bonificaciones (hasta +10 puntos extra)

| Criterio | Puntos |
|----------|--------|
| Export/import del canvas a JSON (guardar y cargar dibujos) | +3 |
| Sistema de undo/redo (deshacer/rehacer) | +3 |
| Zoom y pan (acercar/alejar y desplazar el canvas) | +3 |
| Modo oscuro (tema claro/oscuro) | +2 |
| Atajos de teclado (S=select, D=draw, R=rect, etc.) | +2 |

*Nota: La bonificacion maxima es +10 puntos aunque la suma de los extras supere ese valor.*

---

## Penalizaciones

| Criterio | Descuento |
|----------|-----------|
| La aplicacion no ejecuta sin errores | -10 |
| Codigo sin comentarios ni documentacion | -5 |
| Uso de bibliotecas externas no autorizadas (aparte de nicegui) | -5 |
| Codigo copiado de otro estudiante | -100 |
| Variables o funciones con nombres no descriptivos | -3 |
| No manejar errores basicos (division por cero, indices fuera de rango) | -3 |

---

## Notas importantes

- Cada tarea se evalua de forma independiente. No es necesario completar todas.
- Se valora la calidad del codigo tanto como la funcionalidad.
- Documenta tus decisiones de diseno con comentarios.
- Si implementas algo diferente a lo pedido, explica por que en un comentario.
- Pueden trabajar de forma individual o en parejas.
- Fecha de entrega: consultar con el profesor.
